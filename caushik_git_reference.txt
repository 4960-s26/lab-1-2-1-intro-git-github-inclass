git status
Shows the current state of the repo (modified/staged/untracked files).

git add <file>
Stages a file so it will be included in the next commit.

git add .
Stages all changes in the current directory.

git commit -m "message"
Creates a commit (snapshot) of staged changes.

git push
Uploads your commits to GitHub (remote repo).

git pull
Downloads remote changes and merges them into your local branch.

git fetch
Downloads remote updates but does not merge automatically.

git branch
Lists branches (or creates a new branch with git branch <name>).

git switch <branch>
Switches to another branch.

git merge <branch>
Merges another branch into your current branch.

git log
Shows commit history.
